Code Examples

PHP

To use the api from PHP via curl you can try this example:


$service_url = 'https://edify-cl-qa.herokuapp.com/api/v1/curricula?all=false&from=0&size=10';
$curl = curl_init($service_url);

$request_headers = array();
$request_headers[] = 'Authorization: 05473d65c0e39175cd24838948d56a66e1ec9173790362838c6fd302b0a6a46e';
curl_setopt($curl, CURLOPT_HTTPHEADER, $request_headers);

$curl_response = curl_exec($curl);
if ($curl_response === false) {
  //Something went wrong!
}
curl_close($curl);
$decoded = json_decode($curl_response);

Javascript

To use the api from Javascript you can try this code:


var xhr = new XMLHttpRequest();
xhr.open('GET', 'https://edify-cl-qa.herokuapp.com/api/v1/curricula?all=false&from=0&size=10');
xhr.onreadystatechange = handler;
xhr.responseType = 'json';
xhr.setRequestHeader('Accept', 'application/json');
xhr.setRequestHeader('Authorization', '05473d65c0e39175cd24838948d56a66e1ec9173790362838c6fd302b0a6a46e');
xhr.send();
function handler() {
  //Handle the response
}
